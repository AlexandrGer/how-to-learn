# Научиться учиться
Проект представляет собой лендинг, для проверки и закрепления полученных знаний, при прохождении обучения у Яндекс.Практикума

## Технологии:
* Flexbox вёрстка
* CSS анимации и трансформации
* Организация файловой структуры по БЭМ - Nested
